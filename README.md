# Smart Railway BLE Boarding System

An IoT-based automated railway boarding system using BLE, ESP32, smart gates, and a cloud backend.

## Features

- BLE passenger identification
- Smart automated gates
- Coach occupancy monitoring
- TTE dashboard
- Admin dashboard
- Ticket verification
- Real-time database
- Mobile app

## Hardware

- ESP32
- Servo Motor
- Ultrasonic Sensor
- BLE
- OLED Display

## Software

- Python
- FastAPI
- React
- Firebase/MySQL
- MQTT
